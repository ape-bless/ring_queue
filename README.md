# ring_queue
# 1. 简介
该项目实现了一个同时支持一写一读、多写一读、多写多读的通用、高效的循环队列，可用于消息队列、异步日志的读写等场景。其中的多写、多读，都可以通过编译开关启用、禁止。为了提升性能，我们采用了inplacement new 来减少内存的分配、复制，同时我们每次读、写操作，都会进行字节对齐，来提升内存的访问速度。
队列类名之所以包含一个Compact(紧凑的)，是因为在保证性能的同时，还兼顾到了内存的紧凑性。
这个循环队列一写一读、多写一读部分，已经经过了一款百万DAU的线上项目的验证，可以放心使用，多写多读只是在测试环境压测过。
